# MarketFeeds

## Features

- **Semantic HTML5 Structure**: Uses proper semantic elements for improved accessibility and SEO
- **Responsive Design**: Adapts to desktop, tablet, and mobile views using media queries
- **Modern CSS**: Implements Flexbox, Grid, and animations for a polished look
- **Mobile-First Approach**: Ensures seamless experience across all devices
- **User Registration**: Secure registration system for personalized market insights

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Responsive Design

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## Project Structure

- `index.html`: Main landing page with platform overview
- `register.html`: User registration page
